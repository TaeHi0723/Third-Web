선택자에 대하여

id attribute, class attribute는 CSS가 아니다. HTML이다. 태그에 속성을 줌으로써 선택자를 이용하여 CSS의 효과를 보다 효율적으로 줄 수 있게 하기 위해 사용한다. 선택자 id: #id, class: .class를 사용한다. #과 .을 붙이지 않으면 이름이 id, class인 태그를 선택하게 되며 id, class '값'을 가진 태그를 호출하기 위해서 꼭 붙여야 한다. CSS가 아니다. HTML로 id 혹은 class 값을 가진 태그를 만들 때 사용한다. class와 달리 id의 값은 단 한번만 등장해야 한다. 웹페이지에서 유일무이한 값으로, 중복이 안 된다는 뜻이다. 중복해서 작성해도 동작은 하지만 그렇게 하지 말라고 설명서에 쓰여 있다. 동작 시, 하나의 태그만 적용 되는 것을 볼 수 있는데, id 가 유일하다는것을 이미 가정하고 실행되는 것이라 생각하는 것이 좋다. id는 하나의 태그에 하나의 속성값을 가지고 웹페이지에 한번만 등장할 수 있다. class는 여러개의 태그에 줄 수 있으며, 여러개의 속성값을 가질 수 있다. 속성값은 띄어쓰기로 분리한다. 여러번 등장할 수 있다. ID 선택자를 예외를 두어 구체적인 선택자와 포괄적인 선택자를 구분하여 디자인과 코딩의 효율을 높일 수 있다. id는 특정 태그를 '지정'하는 느낌이고, class는 여러 태그를 'grouping'하는 느낌이다. 여러 효과가 중복되어 있을 경우 같은 레벨의 태그인 경우, 가장 늦은 명령을 따르며. 다른 레벨일 경우 id, class, tag의 순서대로 명령을 따라 효과가 적용된다
