HTML 요소

element
HTML 언어로 작성된 문서의 구성 요소 단어, 문법과 혼용되어 사용 되는 듯 하다 '태그, 속성, 속성값, 값'은 요소의 하위 개념이다 tag, attribute, attribute value, value

태그

tag
무언가(표현, 기능)를 '설명'한다 무언가를 설명할 때 사용하는 HTML의 규칙 즉, 문법이다 HTML을 구성하는 기본 단위이다 앞에 있는 태그를 여는 태그(opening tag), 뒤에 있는 태그를 닫는 태그(closing tag)라고 한다 부모(parent) 태그, 자식(child) 태그

속성

attribute
태그만으로 부족한 정보를 보충해준다
