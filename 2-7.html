CSS코드의 재사용

link 태그로 CSS 파일과 연결하여, 웹브라우저가 css 파일을 다운로드 받아서 웹페이지에 적용하도록 한다. 재사용성, 사용성, 효율성, 경제성, 속도를 높일 수 있다. 캐싱, 저장의 의미. CSS파일을 다운로드 받으면 비효율적이지만 캐싱이라는 저장을 통해 효율을 높일 수 있다. CSS 파일을 한번 다운 받으면 변경이 있을 때까지, 다운로드하지 않고 요청이 있을 때 미리 다운 받아둔 CSS 파일을 사용한다.
